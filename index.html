Checking GitHub from Git

Do You know that there are 3 ways to interact with github

  • HTTPS
  • SSH
  • GitHub CLI

HTTPS

The use of password to gain access to your remote Repositories that is GitHub from the git command line has been stopped, so how do we access our github Repositories on GitHub?

What has Replaced the use of passwords?

They are 2 types of personal access tokens(pat):

  • fine-grained personal access token
  • personal access token (classic)

Do You Know that Public Repositories can Configure their repo to restrict the access of personal access token (classic)

If you try to use a personal access token (classic) to access resources in an organization that has disabled personal access token (classic) access, your request will fail with a 403 response. instead you should use any of these:

  • GitHub App, or
  • OAuth app,
  • fine-grained personal access token.
